# zsh-theme

Forked from [dragon-zsh-theme](https://github.com/perillamint/dragon-zsh-theme), which is a minimalistic zsh prompt theme for git users.

## Feature

*   Shows full path of current directory
*   ISO 8601 Time stamp
*   Shows git status/prompt info
*   Indicates local/ssh
